# 🧪 Ilumionate Beta Tester Guide

Thank you for joining the Ilumionate beta! This guide will help you get the most out of your testing experience and provide valuable feedback.

## 🚀 Getting Started

### 1. Installation
- Download from TestFlight (link provided via email)
- Install on your primary iPhone or iPad
- Ensure you have iOS 18+ (app currently requires iOS 26 beta)

### 2. First Launch
- Complete the 6-step onboarding flow
- Pay attention to clarity and ease of understanding
- Try the 3-minute welcome session
- Note any confusing steps or unclear instructions

### 3. Initial Exploration
- Browse the session library (12 built-in sessions)
- Try different session types:
  - **Relaxation**: `deep_relaxation_session`, `evening_wind_down`
  - **Focus**: `focus_enhancement_session`, `morning_energizer`
  - **Meditation**: `hypnagogic_drift_session`
  - **Advanced**: `bilateral_wave_pattern`, `focused_trance_bilateral`

## 🎯 Key Testing Areas

### A. Onboarding Experience (Priority: HIGH)
**Test Steps:**
1. Delete and reinstall the app
2. Go through onboarding without skipping
3. Complete the welcome session
4. Rate the experience 1-10

**What to Look For:**
- Is the explanation of light therapy clear?
- Do the preparation tips make sense?
- Does the welcome session work smoothly?
- Any confusing or overwhelming parts?

### B. Session Performance (Priority: HIGH)
**Test Steps:**
1. Try 3-5 different pre-built sessions
2. Test play/pause/stop controls
3. Let at least one session complete fully
4. Try both regular and bilateral modes

**What to Look For:**
- Smooth light patterns without flickering
- Accurate timing and progress
- App remains responsive during sessions
- No crashes or freezes
- Battery usage seems reasonable

### C. Audio Integration (Priority: MEDIUM)
**Test Steps:**
1. Import an audio file (.m4a or .mp3)
2. Wait for AI analysis to complete
3. Generate a custom session
4. Play the generated session with your audio

**What to Look For:**
- Analysis completes successfully
- Generated session seems appropriate for your audio
- Audio and light sync properly
- No distortion or audio issues

### D. User Interface (Priority: MEDIUM)
**Test Steps:**
1. Navigate all main screens
2. Try both light and dark mode
3. Test on different device orientations
4. Use all buttons and controls

**What to Look For:**
- Text is readable and clear
- Buttons respond immediately
- Colors and themes are pleasing
- No visual glitches or layout issues

## 📱 Device-Specific Testing

### iPhone Users
- Test in portrait and landscape
- Try with and without headphones
- Test during phone calls (app should pause gracefully)
- Check Control Center integration

### iPad Users
- Test landscape orientation primarily
- Use split-screen if desired
- Check larger text and touch targets
- Verify keyboard interactions

## 🔍 Advanced Testing Scenarios

### Memory & Performance Testing
1. **Long Session Test**: Run a 20+ minute session
2. **Multiple Sessions**: Switch between 3-4 sessions quickly
3. **Background/Foreground**: App should handle being backgrounded
4. **Low Battery**: How does the app behave with <20% battery?

### Edge Cases
1. **Interruptions**: What happens with notifications, calls, alarms?
2. **Storage Issues**: Import many audio files to test storage limits
3. **Network Issues**: Test with poor/no internet connection
4. **Device Limits**: Push the app with many playlists and files

### Accessibility Testing (If Applicable)
1. **VoiceOver**: Navigate with screen reader enabled
2. **Large Text**: Test with largest accessibility text sizes
3. **Reduced Motion**: Verify respect for motion sensitivity settings
4. **High Contrast**: Test with system high contrast enabled

## 📊 What to Report

### Critical Issues (Report Immediately)
- App crashes or becomes unresponsive
- Data loss (sessions, playlists disappear)
- Security concerns or unexpected permissions
- Inability to complete core functions

### Important Issues (Report Within 24 Hours)
- Performance problems (lag, slow responses)
- Audio sync issues or distorted sound
- UI problems (text unreadable, buttons don't work)
- Battery draining unusually fast

### Enhancement Feedback (Report Weekly)
- Feature requests or improvements
- User experience suggestions
- Design feedback
- Missing functionality
